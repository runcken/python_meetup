# python_meetup
Сервис PythonMeetup-бот для подписки на конференции об изменении конференций и презентаций. Доступна отправка вопросов спикеру, для спикера соотвественно возможность их получения. Также реализована возможность познакомиться с другими пользователями бота по интересам и возможность поддерать митап. Для редактирования конференций и презентаций доступна админка на Django.

## Запуск

Для запуска сайта вам понадобится Python 3.12.

Скачайте код с GitHub. Установите зависимости:

```sh
pip install -r requirements.txt
```

Создайте базу данных SQLite

```sh
python3 manage.py migrate
```

Запустите разработческий сервер(127.0.0.1:8000/admin)

```
python3 manage.py runserver
```

Для доступа к админке(127.0.0.1:8000/admin)

```
python3 manage.py createsuperuser
```

Для запуска бота
```
python3 manage.py runbot
```

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` рядом с `manage.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступны переменные:
- `TG_TOKEN` — токен вашего телеграм бота.


Меню бота для пользователя:
- Вопрос спикеру
- Программа
- Нетворкинг
- Поддержать митап

Меню бота для спикера:
- Вопрос спикеру
- Программа
- Нетворкинг
- Мои вопросы
- Поддержать митап

Принадлежность аккаунта к спикеру определяется в админке.

Доступны следующие команды, расширяющие функционал бота:
- /start — описание и главное меню
- /help — помощь
- /update — обновить меню
- /subscribe — подписаться на уведомления
- /unsubscribe — отписаться от уведомлений
- /settings — настройки уведомлений
- /my_questions — для спикеров: посмотреть вопросы
